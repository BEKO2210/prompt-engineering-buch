# Kapitel 2: Was sind Prompts wirklich?

## Die Illusion der Intelligenz

Hier ist die Wahrheit, die niemand dir sagt: KIs verstehen nichts. Sie berechnen Wahrscheinlichkeiten. Wenn du "Schreib mir einen lustigen Text" eingibst, errät die KI Wort für Wort, was wahrscheinlich als Nächstes kommt.

Das klingt enttäuschend, ist aber der Schlüssel zum Verständnis. Wenn du weißt, wie die KI "denkt", kannst du sie steuern.

## Was ist ein Prompt wirklich?

Ein Prompt ist eine Instruktion. Nichts mehr, nichts weniger. Du sagst der KI:
1. Was sie tun soll
2. Wie sie es tun soll
3. In welchem Format

Das ist alles. Keine Zauberei, keine Geheimnisse.

## Der Unterschied zwischen Fragen und Prompts

Die meisten Menschen fragen die KI. Das ist der Fehler.

**Frage (schlecht):**
```
Was ist Marketing?
```

**Prompt (gut):**
```
Erkläre Marketing einem 12-Jährigen. 
Nutze ein Beispiel aus dem Alltag.
Maximal 100 Wörter.
```

Der Unterschied? Die Frage lässt der KI alle Entscheidungen. Der Prompt gibt klare Parameter vor.

## Warum die meisten Prompts scheitern

Ich habe Tausende Prompts gesehen. Die häufigsten Fehler:

1. **Zu vage**: "Mach was Gutes"
2. **Zu komplex**: Anweisungen über drei Absätze verteilt
3. **Kein Kontext**: Die KI weiß nicht, für wen sie schreibt
4. **Kein Format**: Die KI entscheidet selbst, wie die Ausgabe aussieht

Wenn dein Prompt mehr als eine Sache gleichzeitig verlangt, wird das Ergebnis schlecht. Punkt.

## Die drei Ebenen eines Prompts

### Ebene 1: Die Anweisung
Was soll passieren? Schreiben, analysieren, übersetzen?

### Ebene 2: Die Parameter
Wie soll es passieren? Ton, Länge, Format, Zielgruppe?

### Ebene 3: Der Kontext
Für wen? Wozu? Mit welchem Hintergrundwissen?

Jede Ebene, die du weglässt, lässt der KI Spielraum. Manchmal ist das gewollt. Meistens nicht.

## Prompts sind keine Konversationen

Viele behandeln KI-Chats wie Gespräche. Das ist ineffizient. Ein guter Prompt ist eine Spezifikation, kein Smalltalk.

**Falsch:**
```
Hi, ich brauche Hilfe bei einem Blogartikel. 
Es geht um Kaffee. 
Kannst du mir da was schreiben?
```

**Richtig:**
```
Schreibe einen Blogartikel (800 Wörter) über die 
Gesundheitsvorteile von Kaffee. Zielgruppe: 
Gesundheitsbewusste 30-45-Jährige. Ton: 
wissenschaftlich fundiert, aber verständlich. 
Struktur: Einleitung, 3 Hauptpunkte, Fazit.
```

## Die Macht der Präzision

Je präziser dein Prompt, desto besser das Ergebnis. Das ist keine Überraschung – es ist Logik.

Stell dir vor, du gibst einem Autor den Auftrag: "Schreib ein Buch." Das Ergebnis wird nicht das sein, was du wolltest.

Gibst du ihm stattdessen: "Schreibe einen 300-seitigen Thriller über einen forensischen Buchhalter in Berlin. Zielgruppe: Fans von Stieg Larsson. Stil: düster, schnell, dialoglastig." – dann weiß er, was er tun muss.

Mit KIs ist das genauso.

## Deine erste Übung

Nimm einen Prompt, den du kürzlich geschrieben hast. Schreib ihn neu. Füge hinzu:
- Eine klare Rolle (wer soll antworten?)
- Einen Kontext (für wen? wozu?)
- Ein Format (wie soll es aussehen?)
- Eine Aufgabe (was genau soll passieren?)

Vergleiche die Ergebnisse. Du wirst den Unterschied sofort sehen.
