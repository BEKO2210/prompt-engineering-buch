# Übersicht: ROKA-Rahmen

```
┌────────────────────────────────────────────────────────────┐
│                    ROKA-RAHMEN                             │
│         Rolle · Organisation · Konkrete Aufgabe            │
│                    · Ausgabeformat                         │
└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│  R - ROLLE                                                 │
│  ═══════════                                               │
│                                                            │
│  Als [spezifische Rolle]                                   │
│  mit [Erfahrung/Kontext]                                   │
│  für [Zielgruppe/Audienz]...                               │
│                                                            │
│  Beispiel:                                                 │
│  Als SEO-Experte mit 10 Jahren Erfahrung                   │
│  im B2B-Bereich für ein Team von Einsteigern...            │

                              │
                              ▼

│  ═══════════════════════════                               │

│  • Unsere Constraints sind [Einschränkungen]               │
│  • Bisher haben wir [Status Quo]                           │
│                                                            │

                              │
                              ▼
┌────────────────────────────────────────────────────────────┐
│  K - KONKRETE AUFGABE                                      │
│  ═════════════════════                                     │

│  • Strukturiere  • Übersetze  • Fasse zusammen             │
│  • Vergleiche  • Empfehle                                  │
│                                                            │
│  Beispiel:                                                 │
│  Erstelle eine Content-Strategie für 3 Monate,             │
│  die zu 20% mehr Leads führt.                              │

                              │
                              ▼

│  A - AUSGABEFORMAT                                         │
│  ═════════════════                                         │
│                                                            │

│  Länge: [Wörter/Sätze/Zeichen]                             │
│  Ton: [formell/locker/professionell]                       │
│  Stil: [kurz/ausführlich/bullet-points]                    │
│                                                            │
│  Beispiel:                                                 │
│  - Tabelle mit 3 Spalten                                   │
│  - Maximal 10 Zeilen                                       │
│  - Ton: pragmatisch                                        │
└────────────────────────────────────────────────────────────┘
```

## ROKA im Überblick

| Element | Frage | Was gehört rein? |
|---------|-------|------------------|
| **R**olle | Wer antwortet? | Spezifische Rolle, Erfahrung, Zielgruppe |
| **O**rganisation | Was ist der Kontext? | Wer, Was, Warum, Constraints |
| **K**onkrete Aufgabe | Was soll passieren? | Verb + Objekt + Parameter + Ziel |
| **A**usgabeformat | Wie soll es aussehen? | Struktur, Länge, Ton, Stil |

## ROKA-Template zum Kopieren

```
ROLLE:
Als [Rolle] mit [Erfahrung] für [Zielgruppe]...

ORGANISATION:
- [Kontextpunkt 1]
- [Kontextpunkt 2]
- [Kontextpunkt 3]

KONKRETE AUFGABE:
[Verb] [Objekt] [Parameter] [Ziel]

AUSGABEFORMAT:
- [Struktur]
- [Länge]
- [Ton]
- [Stil]
```

## Häufige Fehler im ROKA-Rahmen

| Säule | Fehler | Korrektur |
|-------|--------|-----------|
| Rolle | Zu allgemein | Spezifizieren |
| Organisation | Zu viel | Auf 3-5 Punkte reduzieren |
| Aufgabe | Unklar | Verb + Objekt + Parameter |
| Format | Fehlend | Immer definieren |
