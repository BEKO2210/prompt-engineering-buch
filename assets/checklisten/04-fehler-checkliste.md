# Checkliste: Fehler vermeiden

## Die 10 häufigsten Fehler

### 1. Zu vage sein
**Erkennungszeichen:**
- [ ] Prompt enthält Wörter wie "etwas", "was", "irgendwie"
- [ ] Keine spezifischen Parameter
- [ ] Die KI muss raten, was gemeint ist

**Lösung:**
- Konkrete Verben nutzen
- Parameter definieren (Länge, Format, Ton)
- Beispiele geben

**Vorher:** "Schreib was über Marketing."
**Nachher:** "Schreibe einen LinkedIn-Post (150 Wörter) über Content-Marketing für B2B."

---

### 2. Mehrere Aufgaben in einem Prompt
**Erkennungszeichen:**
- [ ] Prompt enthält "und" mehrfach
- [ ] Verschiedene Ausgabeformate gewünscht
- [ ] Prompt ist länger als 200 Wörter

**Lösung:**
- Aufteilen in separate Prompts
- Ein Prompt = Eine Aufgabe
- Workflow statt Monolith

**Vorher:** "Schreib einen Blogartikel, übersetze ihn, erstelle Social Posts..."
**Nachher:** 4 separate Prompts

---

### 3. Keinen Kontext geben
**Erkennungszeichen:**
- [ ] Prompt startet direkt mit der Aufgabe
- [ ] Keine Informationen über Zielgruppe
- [ ] Keine Informationen über Ziele

**Lösung:**
- 3-5 Kontextpunkte hinzufügen
- Wer, Was, Warum, Wozu klären

**Vorher:** "Bewerte diese Idee."
**Nachher:** "Bewerte diese Idee im Kontext: Zielgruppe X, Markt Y, Budget Z..."

---

### 4. Den Ton nicht definieren
**Erkennungszeichen:**
- [ ] Ergebnis klingt nicht wie gewünscht
- [ ] Zu formell oder zu locker
- [ ] Passt nicht zur Zielgruppe

**Lösung:**
- Ton explizit beschreiben
- Beispiele für den gewünschten Ton geben

**Beispiele:**
- "Ton: professionell, aber nicht steif"
- "Ton: wie ein guter Freund, der dir einen Tipp gibt"
- "Ton: knallhart, keine Ausreden"

---

### 5. Zu komplizierte Prompts
**Erkennungszeichen:**
- [ ] Prompt länger als 300 Wörter
- [ ] Mehr als 5 Anforderungen
- [ ] Widersprüchliche Anweisungen

**Lösung:**
- Kürzen auf das Wesentliche
- Priorisieren: Was ist wirklich wichtig?
- Aufteilen in mehrere Prompts

---

### 6. Keine Beispiele geben
**Erkennungszeichen:**
- [ ] Ergebnis entspricht nicht dem gewünschten Format
- [ ] Stil ist inkonsistent
- [ ] Qualität schwankt

**Lösung:**
- Few-Shot Prompting nutzen
- 2-5 Beispiele für gewünschte Ausgabe

---

### 7. Das Format vergessen
**Erkennungszeichen:**
- [ ] Ergebnis hat falsche Länge
- [ ] Falsche Struktur
- [ ] KI entscheidet selbst über Format

**Lösung:**
- Format explizit definieren
- Struktur, Länge, Ton angeben

---

### 8. An die falsche KI stellen
**Erkennungszeichen:**
- [ ] KI kann Aufgabe nicht erfüllen
- [ ] Halluzinationen bei Fakten
- [ ] Falsches Tool für die Aufgabe

**Lösung:**
- Richtiges Tool wählen
- ChatGPT für Allgemeines
- Claude für lange Texte
- Perplexity für Recherche
- Spezialisierte Tools für Spezialaufgaben

---

### 9. Nicht iterieren
**Erkennungszeichen:**
- [ ] Erster Versuch = Endresultat
- [ ] Keine Verbesserungsversuche
- [ ] Frustration über schlechte Ergebnisse

**Lösung:**
- Erster Prompt = Entwurf
- Review und Iteration planen
- "Kürzer", "Detaillierter", "Anderer Ton"

---

### 10. Alles der KI überlassen
**Erkennungszeichen:**
- [ ] Prompt ist zu allgemein
- [ ] Keine eigenen Inputs
- [ ] Generische Ergebnisse

**Lösung:**
- Eigenes Wissen einbringen
- Kontext liefern
- Richtung vorgeben

---

## Fehler-Notizbuch

Für jeden Fehler, den du machst:

```
Datum: [Datum]
Fehler: [Was ist passiert?]
Prompt: [Original-Prompt]
Ergebnis: [Was kam raus?]
Warum schlecht: [Analyse]
Korrektur: [Wie wurde es gefixt?]
Gelernt: [Was nehme ich mit?]
```

---

## Schnell-Fehler-Check

Vor dem Absenden:
- [ ] Ist es spezifisch?
- [ ] Ist es eine Aufgabe?
- [ ] Ist der Kontext klar?
- [ ] Ist das Format definiert?

Wenn alle Haken: Absenden.
Wenn nicht: Korrigieren.

---

*Jeder Fehler ist eine Lernchance. Nutze sie.*
