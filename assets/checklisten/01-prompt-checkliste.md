# Checkliste: Prompt vor dem Absenden

## Die 10-Punkte-Prüfung

Vor jedem Prompt, gehe diese Liste durch:

### 1. Rolle definiert?
- [ ] Wer antwortet? (spezifisch, nicht generisch)
- [ ] Welche Erfahrung hat diese Rolle?
- [ ] Für wen antwortet sie? (Zielgruppe)

**Beispiel gut:** "Als SEO-Experte mit 10 Jahren Erfahrung im E-Commerce für ein Team von Einsteigern..."

**Beispiel schlecht:** "Als Experte..."

### 2. Kontext klar?
- [ ] Wer ist der Empfänger?
- [ ] Was ist das Ziel?
- [ ] Was ist der Hintergrund?
- [ ] Gibt es Einschränkungen?

### 3. Aufgabe spezifisch?
- [ ] Eindeutiges Verb (schreibe, analysiere, erstelle)
- [ ] Klares Objekt (was genau?)
- [ ] Parameter definiert (wie viel, wie lange, wie detailliert)
- [ ] Ziel angegeben (was soll erreicht werden?)

### 4. Format definiert?
- [ ] Struktur (Liste, Tabelle, Fließtext)
- [ ] Länge (Wörter, Sätze, Zeichen)
- [ ] Ton (formell, locker, professionell)
- [ ] Stil (kurz, ausführlich, bullet-points)

### 5. Eine Aufgabe pro Prompt?
- [ ] Nur eine Hauptaufgabe?
- [ ] Keine mehrfachen Anforderungen?

**Falsch:** "Schreib einen Blogartikel, übersetze ihn, erstelle Social Posts..."

**Richtig:** Teile auf in mehrere Prompts.

### 6. Beispiele gegeben (wenn nötig)?
- [ ] Bei Klassifikation: Beispiele für jede Kategorie
- [ ] Bei Format: Beispiel der gewünschten Ausgabe
- [ ] Bei Stil: Beispieltexte

### 7. Negative Constraints definiert?
- [ ] Was soll vermieden werden?
- [ ] Was ist explizit nicht gewollt?

**Beispiel:** "Vermeide Füllwörter wie 'wirklich', 'einfach', 'natürlich'."

### 8. Realistische Erwartungen?
- [ ] Kann die KI das leisten?
- [ ] Ist die Anfrage zu komplex für einen Prompt?
- [ ] Sind die Parameter widerspruchsfrei?

### 9. Richtige KI gewählt?
- [ ] ChatGPT für allgemeine Aufgaben
- [ ] Claude für lange Texte
- [ ] Perplexity für Recherche
- [ ] Spezialisierte Tools für spezielle Aufgaben

### 10. Bereit zu iterieren?
- [ ] Erster Versuch = Entwurf
- [ ] Review-Plan vorhanden
- [ ] Verbesserungsrichtung klar

---

## Schnell-Check (30 Sekunden)

Wenn du nur 30 Sekunden hast:

1. [ ] Rolle drin?
2. [ ] Kontext klar?
3. [ ] Aufgabe spezifisch?
4. [ ] Format definiert?

Wenn alle 4 Haken: Absenden.
Wenn nicht: Ergänzen.

---

## Nach dem Absenden

- [ ] Ergebnis bewertet (1-5)
- [ ] Notiert, was gut/schlecht war
- [ ] Iterationsidee notiert
- [ ] Für Prompt-Bibliothek markiert (falls gut)

---

*Speichere diese Checkliste. Nutze sie. Deine Prompts werden besser.*
