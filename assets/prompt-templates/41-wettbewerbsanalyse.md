# Prompt-Template 41: Wettbewerbsanalyse

## Zweck
Strukturierte Analyse von Mitbewerbern und Marktposition.

## Prompt
```
ROLLE: Als Strategie-Berater...

KONTEXT:
Markt: [Branche/Segment]
Eigenes Angebot: [Was bieten wir an?]
Wettbewerber: [3-5 Hauptkonkurrenten]
Ziel: [Was wollen wir herausfinden?]

AUFGABE: Erstelle eine Wettbewerbsanalyse.

FORMAT:
Pro Wettbewerber:
- Überblick: Positionierung und Größe
- Stärken: Was machen sie gut?
- Schwächen: Wo sind sie verwundbar?
- Preise: Pricing-Strategie
- Zielgruppe: Wen adressieren sie?
- Marketing: Kanäle und Messaging
- Differentiation: Was unterscheidet sie?

Fazit: Unsere Positionierung und Chancen.
```

## Variationen
- Für neue Märkte: Markteintrittsstrategie
- Für Features: Funktions-Vergleich (Feature Matrix)
- Für Pricing: Preis-Leistungs-Analyse
