# Prompt Template: Kreatives Schreiben

```
ROLLE:
Als [Autor/Copywriter/Kreativschreiber] mit 
Fokus auf [Genre/Stil]...

ORGANISATION:
- Format: [Blog/Roman/Kurzgeschichte/Gedicht]
- Thema: [Was soll es behandeln?]
- Zielgruppe: [Wer soll es lesen?]
- Ton: [humorvoll/dramatisch/inspirierend/düster]
- Länge: [X Wörter/Sätze]
- Constraints: [Was muss enthalten sein? Was nicht?]

KONKRETE AUFGABE:
Schreibe [Format] zum Thema [Thema].

AUSGABEFORMAT:
Je nach Format:

Für Blog:
- Überschrift (catchy)
- Einleitung (Hook)
- Hauptteil (Struktur vorgeben)
- Fazit (CTA)

Für Geschichte:
- Setting (Wo/Wann?)
- Charaktere (Wer?)
- Plot (Was passiert?)
- Konflikt (Spannung)
- Auflösung (Ende)

Für Gedicht:
- Form (frei/gebunden)
- Reimschema (falls gewünscht)
- Bildsprache (Metaphern)

Stil-Vorgaben: [konkrete Anweisungen]
```
