# Prompt-Template 26: Webinar-Skript

## Zweck
Strukturierte Skripte für überzeugende Online-Präsentationen.

## Prompt
```
ROLLE: Als Webinar-Moderator mit Erfahrung in der Conversion...

KONTEXT:
Thema: [Thema]
Zielgruppe: [Teilnehmer]
Ziel: [Bildung/Verkauf/Lead-Generierung]
Dauer: [30/45/60 Minuten]

AUFGABE: Schreibe ein komplettes Webinar-Skript.

FORMAT:
1. Opening (5 Min): Hook + Agenda + Credibility
2. Problem (10 Min): Schmerzpunkte verstärken
3. Lösung (15 Min): Framework/Methodik vorstellen
4. Case Study (10 Min): Soziales Beweismaterial
5. Angebot (10 Min): Produkt/Dienstleistung vorstellen
6. Q&A (10 Min): Einwände adressieren
7. Closing (5 Min): Dringlichkeit + CTA

Inklusive: Sprechernotizen für jeden Abschnitt.
```

## Variationen
- Für Verkauf: Mehr Zeit für Angebot und Garantien
- Für Onboarding: Interaktive Elemente und Übungen
- Für Thought Leadership: Weniger Pitch, mehr Insights
