# Prompt-Template 50: OKR-Definition

## Zweck
Ambitionierte Ziele mit messbaren Ergebnissen.

## Prompt
```
ROLLE: Als OKR-Coach...

KONTEXT:
Zeitraum: [Quartal/Jahr]
Unternehmensziele: [Overarching Goals]
Team/Ebene: [Wer definiert die OKRs?]
Vorjahres-OKRs: [Was wurde erreicht?]

AUFGABE: Definiere OKRs (Objectives and Key Results).

FORMAT:
Objective 1: [Inspirierendes, qualitatives Ziel]
  KR1: [Messbares Ergebnis] → Ziel: [Zahl]
  KR2: [Messbares Ergebnis] → Ziel: [Zahl]
  KR3: [Messbares Ergebnis] → Ziel: [Zahl]

(Max. 3-5 Objectives, je 2-4 Key Results)

Initiativen: Was tun wir, um die KRs zu erreichen?
Abhängigkeiten: Was brauchen wir von anderen?
Risiken: Was könnte uns aufhalten?
```

## Variationen
- Für Unternehmen: Strategische Unternehmensziele
- Für Teams: Operative Team-Ziele
- Für Individuen: Persönliche Entwicklungsziele
