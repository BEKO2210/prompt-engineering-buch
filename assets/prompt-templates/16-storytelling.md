# Prompt Template: Storytelling

```
ROLLE:
Als Storyteller/Copywriter mit Fokus auf 
[emotionale/wirkungsorientierte] Narrative...

ORGANISATION:
- Zweck: [Marketing/Präsentation/Intern]
- Kernbotschaft: [Was soll hängen bleiben?]
- Zielgruppe: [Wer soll die Geschichte hören?]
- Kontext: [Wo wird sie erzählt?]
- gewünschte Emotion: [Inspiration/Mitgefühl/Spannung]

KONKRETE AUFGABE:
Entwickle eine Geschichte für [Zweck].

Elemente:
- Protagonist: [Wer?]
- Problem: [Was steht auf dem Spiel?]
- Reise: [Was passiert?]
- Wendepunkt: [Wie ändert sich alles?]
- Auflösung: [Wie endet es?]

AUSGABEFORMAT:
1. Setup (Wer ist der Protagonist?)
2. Konflikt (Was muss überwunden werden?)
3. Eskalation (Was verschlimmert die Lage?)
4. Wendepunkt (Der Moment der Veränderung)
5. Auflösung (Das Happy End oder die Lehre)
6. Takeaway (Was sollen die Zuhörer mitnehmen?)

Länge: [kurz/mittel/lang]
Ton: [dramatisch/inspirierend/unterhaltsam]
```
