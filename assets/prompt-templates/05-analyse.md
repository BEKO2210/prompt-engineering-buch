# Prompt Template: Analyse

```
ROLLE:
Als [Analyst/Experte] mit Fokus auf [Bereich]...

ORGANISATION:
- Ausgangsmaterial: [Text/Daten/Transkript]
- Kontext: [Woher kommt das Material? Wer hat es erstellt?]
- Ziel der Analyse: [Was soll herausgefunden werden?]
- Zielgruppe der Analyse: [Für wen ist das Ergebnis?]

KONKRETE AUFGABE:
Analysiere das folgende Material auf [Aspekt 1, Aspekt 2, Aspekt 3]:

[MATERIAL HIER EINFÜGEN]

AUSGABEFORMAT:
- Zusammenfassung: Max. 3 Sätze
- Key Findings: 3-5 Punkte mit Begründung
- Detaillierte Analyse: Pro Aspekt ein Abschnitt
- Handlungsempfehlungen: Priorisiert (high/medium/low)
- Offene Fragen: Was wurde nicht geklärt?

Struktur: Übersichtlich, mit Zwischenüberschriften
Ton: Sachlich, objektiv
Länge: Angemessen zum Input (max. 20% der Originallänge für Zusammenfassungen)
```
