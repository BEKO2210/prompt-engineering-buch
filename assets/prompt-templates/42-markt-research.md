# Prompt-Template 42: Markt-Research

## Zweck
Datenbasierte Einblicke in Märkte und Trends.

## Prompt
```
ROLLE: Als Marktforscher...

KONTEXT:
Markt: [Branche/Region]
Zielgruppe: [Wen untersuchen wir?]
Fragestellung: [Was wollen wir wissen?]
Datenquellen: [Verfügbare Daten]

AUFGABE: Strukturiere eine Markt-Research-Studie.

FORMAT:
1. Executive Summary: Key Findings (3-5 Punkte)
2. Methodik: Wie wurden Daten erhoben?
3. Marktübersicht: Größe, Wachstum, Trends
4. Zielgruppen-Analyse: Segmente und Bedürfnisse
5. Wettbewerbslandschaft: Marktanteile und Player
6. Trends: Technologisch, regulatorisch, sozial
7. Chancen und Risiken: SWOT-ähnliche Analyse
8. Empfehlungen: Handlungsempfehlungen
```

## Variationen
- Für Produkte: Product-Market-Fit analysieren
- Für Expansion: Neue Regionen/Märkte bewerten
- Für Investoren: Marktpotenzial quantifizieren
