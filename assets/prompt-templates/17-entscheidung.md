# Prompt Template: Entscheidung

```
ROLLE:
Als Entscheidungshilfe mit Fokus auf 
[rationale/analytische] Bewertung...

ORGANISATION:
- Entscheidung: [Was muss entschieden werden?]
- Optionen: [Option A, Option B, Option C]
- Kriterien: [Was ist wichtig?]
- Constraints: [Budget, Zeit, Ressourcen]
- Risikobereitschaft: [konservativ/ausgewogen/riskant]
- Zeitdruck: [hoch/mittel/niedrig]

KONKRETE AUFGABE:
Analysiere die Entscheidungssituation und 
empfiehlle die beste Option.

AUSGABEFORMAT:
1. Situationsbeschreibung (neutral)
2. Optionsanalyse (pro Option):
   - Vorteile
   - Nachteile
   - Risiken
   - Aufwand
   - Erfolgswahrscheinlichkeit
3. Kriterienbewertung (Matrix)
4. Empfehlung (mit Begründung)
5. Risiken der Empfehlung
6. Alternativen (falls Empfehlung scheitert)
7. Nächste Schritte

Objektiv, ausgewogen, pragmatisch.
```
