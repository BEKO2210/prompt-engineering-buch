# Prompt-Template 22: Landing Page Copy

## Zweck
Konvertierungsstarke Landing Page Texte für Produkte oder Dienstleistungen.

## Prompt
```
ROLLE: Als Conversion-Copywriter mit Fokus auf Landing Pages...

KONTEXT: Wir erstellen eine Landing Page für [Angebot].
Zielgruppe: [Beschreibung]
Hauptnutzen: [Nutzen]
Call-to-Action: [Ziel der Seite]

AUFGABE: Schreibe die komplette Landing Page Copy.

FORMAT:
- Hero Section: Headline (max. 8 Wörter) + Subheadline + CTA-Button
- Problem-Agitation: 3 Schmerzpunkte der Zielgruppe
- Lösung: 3 Key Benefits mit Icons
- Sozialer Beweis: Testimonial-Platzhalter
- Feature-Liste: 5 Features mit Benefits
- FAQ: 3 Einwände adressieren
- Final CTA: Dringlichkeit + Garantie

TON: [überzeugend/vertrauensvoll/direkt]
```

## Beispiel-Ausgabe
**Headline:** Nie wieder Überstunden
**Subheadline:** Automatisiere deine Zeiterfassung in 5 Minuten...

## Variationen
- Für SaaS: Features und Integrationen betonen
- Für Coaching: Transformation und Ergebnisse fokussieren
- Für E-Commerce: Garantien und Rückgabe sicherstellen
