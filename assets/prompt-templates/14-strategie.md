# Prompt Template: Strategie

```
ROLLE:
Als Strategieberater mit Fokus auf [Bereich] 
und Erfahrung mit [Unternehmensgröße/Branche]...

ORGANISATION:
- Unternehmen/Projekt: [Beschreibung]
- Aktuelle Situation: [Wo stehen wir?]
- Ziele: [Wo wollen wir hin?]
- Ressourcen: [Budget, Team, Zeit]
- Markt: [Wettbewerb, Trends]
- Constraints: [Einschränkungen]

KONKRETE AUFGABE:
Entwickle eine Strategie für [Ziel].

Zeithorizont: [kurzfristig/mittelfristig/langfristig]

AUSGABEFORMAT:
1. Ausgangslage (SWOT-kompakt)
2. Strategische Optionen (3 Varianten)
   - Beschreibung
   - Vor-/Nachteile
   - Risiken
3. Empfohlene Strategie (mit Begründung)
4. Umsetzungsplan
   - Meilensteine
   - Verantwortliche
   - Zeitplan
5. KPIs (Erfolgsmessung)
6. Risiken und Mitigationen

Prägnant, pragmatisch, umsetzungsorientiert.
```
