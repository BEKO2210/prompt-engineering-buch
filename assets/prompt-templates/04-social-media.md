# Prompt Template: Social Media Post

```
ROLLE:
Als Social Media Manager für [Marke/Unternehmen] 
mit Fokus auf [Plattform: LinkedIn/Instagram/Twitter]...

ORGANISATION:
- Plattform: [LinkedIn/Instagram/Twitter/Facebook]
- Zielgruppe: [demografisch/psychografisch]
- Content-Pillar: [Themenbereich]
- Aktueller Kontext: [Saisonalität, Trends, Ereignisse]

KONKRETE AUFGABE:
Erstelle einen [Post-Typ: Carousel, Text-Post, Video-Skript] 
zu folgendem Thema: [Thema]

Ziel: [Engagement/Reach/Conversions/Brand Awareness]

AUSGABEFORMAT:
Plattform-spezifisch:

Für LinkedIn:
- Hook (erste 2 Zeilen müssen stoppen)
- Hauptteil (max. 150 Wörter)
- Ein praktischer Takeaway
- Call-to-Action (Frage oder Diskussionsanstoß)
- 3-5 relevante Hashtags

Für Instagram:
- Caption (kurz und prägnant)
- Call-to-Action
- Hashtags (10-15)

Für Twitter/X:
- Max. 280 Zeichen
- Ein Hook
- Ein Takeaway oder Call-to-Action

Ton: [inspirierend/provokant/informativ/unterhaltsam]
```
