# Prompt-Template 43: Persona-Entwicklung

## Zweck
Detaillierte Nutzerprofile für Marketing und Produkt.

## Prompt
```
ROLLE: Als UX-Researcher...

KONTEXT:
Zielgruppe: [Wen beschreiben wir?]
Datenbasis: [Interviews/Analytics/Research]
Anwendung: [Marketing/Product/Sales]

AUFGABE: Entwickle eine detaillierte Persona.

FORMAT:
- Name & Foto: Fiktiver Name und Beschreibung
- Demografie: Alter, Beruf, Einkommen, Standort
- Background: Ausbildung, Karriere, Familie
- Ziele: Was will sie erreichen?
- Schmerzpunkte: Was frustriert sie?
- Verhalten: Wie kauft/sie nutzt sie Produkte?
- Kanäle: Wo ist sie aktiv?
- Zitate: Typische Aussagen
- Einwände: Was könnte sie vom Kauf abhalten?
```

## Variationen
- Für B2B: Firmografie und Buying Center
- Für B2C: Lifestyle und Werte priorisieren
- Für Enterprise: Decision-Making-Prozess
