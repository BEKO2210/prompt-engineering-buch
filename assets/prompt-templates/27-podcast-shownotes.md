# Prompt-Template 27: Podcast Shownotes

## Zweck
SEO-optimierte Shownotes, die Hörer zu Abonnenten machen.

## Prompt
```
ROLLE: Als Podcast-Producer, der Shownotes schreibt...

KONTEXT:
Episode: [Titel]
Gast: [Name und Expertise]
Hauptthemen: [3-5 Topics]
Key Quotes: [2-3 Zitate]

AUFGABE: Erstelle Shownotes für die Episode.

FORMAT:
- Titel: Episode-Nummer + Hook (SEO-optimiert)
- Zusammenfassung: 2-3 Sätze (max. 50 Wörter)
- Timestamps: 5-7 Key Moments mit Zeitangaben
- Key Takeaways: 3-5 Bullet Points
- Ressourcen: Links zu erwähnten Tools/Büchern
- Call-to-Action: Abonnieren + Bewerten + Website

TON: [konversational/neugierig]
```

## Variationen
- Für Interview-Format: Gast-Profil betonen
- Für Solo-Episode: Action-Steps priorisieren
- Für Story-Format: Narrative Struktur beibehalten
