# Prompt Template: Feedback

```
ROLLE:
Als [Manager/Coach/Kollege] mit Fokus auf 
konstruktiver Kommunikation...

ORGANISATION:
- Empfänger: [Name, Rolle, Erfahrungslevel]
- Beziehung: [Direkter Vorgesetzter/Peer/Mentor]
- Situation: [Was ist passiert?]
- Beobachtetes Verhalten: [Spezifisch, objektiv]
- Impact: [Welche Auswirkung hatte es?]
- Ziel des Feedbacks: [Was soll sich ändern?]

KONKRETE AUFGABE:
Formuliere Feedback nach dem SBI-Modell:
- Situation: Wann/Wo?
- Behavior: Was wurde beobachtet?
- Impact: Wie wirkte es sich aus?

AUSGABEFORMAT:
1. Einstieg (den Einstieg erleichtern)
2. SBI-Feedback (Situation, Behavior, Impact)
3. Verbesserungsvorschlag (konkret, umsetzbar)
4. Unterstützungsangebot (wie kann ich helfen?)
5. Zukunftsperspektive (positive Erwartung)

Ton: [konstruktiv/respektvoll/direkt]
Länge: Max. 200 Wörter
Vermeide: Persönliche Kritik, Vage Aussagen
```
