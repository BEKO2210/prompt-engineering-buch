# Prompt Template: Blogartikel

```
ROLLE:
Als erfahrener Content-Marketer für [Branche] 
mit Spezialisierung auf [Thema]...

ORGANISATION:
- Zielgruppe: [demografisch/psychografisch]
- Ziel des Artikels: [informieren/überzeugen/verkaufen]
- Unser USP: [was macht uns einzigartig?]
- Tonality Guidelines: [wie sollen wir klingen?]

KONKRETE AUFGABE:
Schreibe einen Blogartikel (800-1000 Wörter) 
über [Thema] mit folgender Struktur:
1. Einleitung mit Hook (stell ein Problem dar)
2. 3-4 Hauptabschnitte mit Zwischenüberschriften
3. Praktische Tipps/Actionable Advice
4. Fazit mit Call-to-Action

AUSGABEFORMAT:
- Überschrift: SEO-optimiert, max. 60 Zeichen
- Zwischenüberschriften: H2, beschreibend
- Absätze: Max. 3-4 Sätze
- Bullet Points für Listen
- Fachbegriffe: Erklären beim ersten Vorkommen
- Meta-Beschreibung: 150-160 Zeichen
- Ton: [professionell/locker/enthusiastisch]
```
