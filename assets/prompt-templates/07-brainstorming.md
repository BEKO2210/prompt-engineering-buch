# Prompt Template: Brainstorming

```
ROLLE:
Als kreativer [Berater/Stratege/Designer] mit 
Erfahrung in [Bereich]...

ORGANISATION:
- Thema/Problem: [Was soll gelöst werden?]
- Constraints: [Budget, Zeit, Ressourcen]
- Bisherige Ansätze: [Was wurde schon versucht?]
- Zielgruppe/Nutzer: [Für wen wird das entwickelt?]

KONKRETE AUFGABE:
Generiere [Anzahl] Ideen für [Thema/Problem].

Ansätze:
- Konventionell/Pragmatisch
- Kreativ/Wild
- Aus einer anderen Branche übertragen
- Das Gegenteil des Offensichtlichen

AUSGABEFORMAT:
Pro Idee:
- Titel/Name
- Kurzbeschreibung (1 Satz)
- Warum es funktionieren könnte
- Potenzielle Herausforderungen
- Geschätzter Aufwand (low/medium/high)
- Geschätztes Potenzial (low/medium/high)

Am Ende:
- Top 3 Ideen mit Begründung
- Kombinationsmöglichkeiten
- Nächster Schritt (was sollte als erstes getestet werden?)
```
