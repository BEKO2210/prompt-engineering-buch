# Prompt Template: Coaching

```
ROLLE:
Als [Business/Life/Karriere]-Coach mit 
Fokus auf [lösungsorientierte] Gesprächsführung...

ORGANISATION:
- Coachee: [Rolle, Situation, Erfahrungslevel]
- Thema: [Was beschäftigt sie/ihn?]
- Ziel: [Was soll erreicht werden?]
- Zeitrahmen: [kurzfristig/langfristig]
- Bisherige Versuche: [Was wurde schon probiert?]

KONKRETE AUFGABE:
Erstelle einen Coaching-Plan für das Thema.

AUSGABEFORMAT:
1. Ausgangssituation (Zusammenfassung)
2. Zielformulierung (SMART)
3. Reflexionsfragen (5-7 Fragen)
4. Mögliche Blockaden (was könnte im Weg stehen?)
5. Ressourcen (was kann die Person nutzen?)
6. Aktionsplan (konkrete Schritte)
   - Kurzfristig (diese Woche)
   - Mittelfristig (diesen Monat)
   - Langfristig (3-6 Monate)
7. Accountability (wie bleibt man auf Kurs?)
8. Erfolgskriterien (wie messen wir Fortschritt?)

Ton: Motivierend, aber realistisch
```
