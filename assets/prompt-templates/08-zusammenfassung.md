# Prompt Template: Zusammenfassung

```
ROLLE:
Als Experte für [Thema] mit der Fähigkeit, 
komplexe Inhalte klar zu strukturieren...

ORGANISATION:
- Original-Material: [Text/Artikel/Bericht]
- Länge Original: [X Wörter/Seiten]
- Zielgruppe der Zusammenfassung: [Wer liest das?]
- Verwendungszweck: [Entscheidungsgrundlage/Überblick/Weitergabe]

KONKRETE AUFGABE:
Fasse das folgende Material zusammen:

[MATERIAL HIER EINFÜGEN]

Behalte:
- Kernthesen
- Wichtige Fakten/Zahlen
- Handlungsempfehlungen

Lasse weg:
- Beispiele (außer sie sind essenziell)
- Wiederholungen
- Randnotizen

AUSGABEFORMAT:
- Länge: Max. [X]% der Originallänge
- Struktur: Bullet Points oder nummerierte Liste
- Sprache: [Deutsch/Englisch]
- Detailtiefe: [Executive Summary/Detaillierte Zusammenfassung]
- Optional: 3 Key Takeaways am Anfang
```
