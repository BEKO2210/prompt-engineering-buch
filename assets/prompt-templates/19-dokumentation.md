# Prompt Template: Dokumentation

```
ROLLE:
Als Technical Writer mit Fokus auf 
[klare/verständliche] Dokumentation...

ORGANISATION:
- Thema: [Was soll dokumentiert werden?]
- Zielgruppe: [Anfänger/Fortgeschrittene/Experten]
- Kontext: [Produkt, System, Prozess]
- Format: [Wiki/Handbuch/API-Doc/FAQ]
- Vorhandene Materialien: [Was gibt es schon?]

KONKRETE AUFGABE:
Erstelle Dokumentation für [Thema].

AUSGABEFORMAT:
1. Übersicht (Was ist das?)
2. Voraussetzungen (Was braucht man?)
3. Schritt-für-Schritt-Anleitung
   - Jeder Schritt nummeriert
   - Screenshots/Beispiele (Platzhalter)
   - Erwartetes Ergebnis pro Schritt
4. Troubleshooting (Häufige Probleme + Lösungen)
5. FAQ (5-10 Fragen)
6. Verwandte Themen (Links)
7. Versionshistorie (wann wurde was geändert?)

Sprache: [Deutsch/Englisch]
Detailtiefe: [hoch/mittel/überblick]
```
