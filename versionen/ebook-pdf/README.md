# E-Book PDF Version

## Hinweise zur PDF-Erstellung

Diese Version enthält das Buch mit professionellem Layout für den PDF-Export.

### Design-Elemente

**Typografie:**
- Überschriften: Bold, klar abgesetzt
- Fließtext: 11pt, Zeilenabstand 1.5
- Code-Blocks: Monospace, Hintergrund hellgrau
- Zitate: Kursiv, Einrückung

**Layout:**
- Seitenränder: 2.5cm
- Kopfzeile: Kapiteltitel
- Fußzeile: Seitenzahl
- Kapitelanfang: Neue Seite

**Farbschema:**
- Primärfarbe: #2C3E50 (Dunkelblau)
- Akzentfarbe: #E74C3C (Rot)
- Hintergrund: #FFFFFF (Weiß)
- Code-Hintergrund: #F8F9FA

### Inhaltsverzeichnis

Automatisch generiert mit Seitenzahlen.

### Kapitel-Struktur

Jedes Kapitel enthält:
- Kapiteltitel (H1)
- Einleitender Absatz
- Hauptteil mit Zwischenüberschriften
- Zusammenfassung (falls sinnvoll)
- Übungen/Action Items (wo angebracht)

### Besondere Elemente

**Info-Boxen:**
┌─────────────────────────────────────┐
│  💡 TIPP                            │
│  Kurze, praktische Hinweise         │
└─────────────────────────────────────┘

**Warnungen:**
┌─────────────────────────────────────┐
│  ⚠️ ACHTUNG                         │
│  Häufige Fehler vermeiden           │
└─────────────────────────────────────┘

**Beispiele:**
┌─────────────────────────────────────┐
│  📋 BEISPIEL                        │
│  Konkrete Prompt-Beispiele          │
└─────────────────────────────────────┘

### Druck-Spezifikationen

- Format: A4
- Farbmodus: RGB (für Bildschirm), CMYK (für Druck)
- Auflösung: 300dpi für Bilder
- Schriften: Eingebettet

---

*Nutze Adobe InDesign, Canva oder LaTeX für die professionelle PDF-Erstellung.*
