# E-Book EPUB Version

## Hinweise zur EPUB-Erstellung

Diese Datei enthält den vollständigen Buchinhalt in einer für EPUB-Reader optimierten Struktur.

### Struktur

1. **Einleitung** - Warum Prompt Engineering wichtig ist
2. **Was sind Prompts** - Grundlagen und Konzepte
3. **Die Anatomie eines guten Prompts** - Die 4 Säulen
4. **Der ROKA-Rahmen** - Die Belkis-Methode
5. **Fortgeschrittene Techniken** - CoT, Few-Shot, System-Prompts
6. **Häufige Fehler** - Was Anfänger falsch machen
7. **Praxisbeispiele** - 20+ konkrete Anwendungsfälle
8. **Tools und Workflows** - Welche KI wofür
9. **Von Anfänger zum Profi** - Der 30-Tage-Lernpfad
10. **Appendix** - 50+ Prompts

### Formatierung

- Einfacher Fließtext ohne komplexe Layouts
- Klare Überschriften-Hierarchie (H1, H2, H3)
- Aufzählungen für bessere Lesbarkeit
- Code-Blocks für Prompt-Beispiele

### Empfohlene EPUB-Einstellungen

- Schriftart: Systemschrift oder serifenlose Schrift
- Zeilenabstand: 1.5
- Rand: Automatisch
- Verzeichnis: Automatisch generieren
